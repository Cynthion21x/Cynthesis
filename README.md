## Setup

### Dependancies
* the python interpreter
* pygame is required ```pip install pygame```
* cx_freeze

<br>

### Installing

Github CLI
``` 
gh repo clone Cynthion21x/Calcuslugs
```

Git
``` 
git clone https://github.com/Cynthion21x/Calcuslugs.git
```

## Usage

to run the program go to the root folder and open main.py
```
python main.py
```

alternatively you can build it using
```
python setup.py build
```
